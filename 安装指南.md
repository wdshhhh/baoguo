# 🚀 菜鸟驿站包裹管理系统 - 完整安装指南

## 📋 系统概述

菜鸟驿站包裹管理系统是一个基于Ruby on Rails + Vue.js的现代化包裹管理解决方案，具备完整的OCR面单识别、AI智能助手、数据分析等功能。

### 核心功能
- 📦 包裹全生命周期管理
- 📸 AI增强OCR面单识别
- 🤖 智能AI助手
- 📊 数据统计与分析
- 📱 移动端适配
- 🔄 批量处理

## 🛠️ 环境要求

### 操作系统
- **推荐**: Ubuntu 20.04+ / CentOS 8+ / macOS 12+
- **支持**: Windows 10+ (WSL2推荐)

### 软件要求
- **Ruby**: 3.0+
- **Rails**: 8.0+
- **Node.js**: 16+
- **SQLite**: 3.8+
- **Tesseract-OCR**: 4.0+
- **ImageMagick**: 7.0+

## 📥 快速安装

### 步骤1: 系统环境准备

#### Ubuntu/Debian系统
```bash
# 更新系统包管理器
sudo apt update && sudo apt upgrade -y

# 安装基础开发工具
sudo apt install -y build-essential curl git libssl-dev libreadline-dev zlib1g-dev

# 安装系统依赖
sudo apt install -y tesseract-ocr tesseract-ocr-chi-sim tesseract-ocr-chi-tra
sudo apt install -y imagemagick sqlite3 libsqlite3-dev
sudo apt install -y redis-server # 可选，用于后台任务

# 验证安装
tesseract --version
convert --version
sqlite3 --version
```

#### CentOS/RHEL系统
```bash
# 安装EPEL仓库
sudo yum install -y epel-release

# 安装基础工具
sudo yum install -y gcc-c++ make curl git openssl-devel readline-devel zlib-devel

# 安装OCR依赖
sudo yum install -y tesseract tesseract-langpack-chi_sim tesseract-langpack-chi_tra
sudo yum install -y ImageMagick sqlite sqlite-devel
sudo yum install -y redis # 可选

# 验证安装
tesseract --version
convert --version
sqlite3 --version
```

#### macOS系统
```bash
# 安装Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装依赖
brew install rbenv ruby-build tesseract imagemagick sqlite3 redis

# 安装中文语言包
brew install tesseract-lang

# 验证安装
tesseract --version
convert --version
sqlite3 --version
```

### 步骤2: 安装Ruby环境

#### 使用rbenv安装Ruby
```bash
# 安装rbenv
curl -fsSL https://github.com/rbenv/rbenv-installer/raw/HEAD/bin/rbenv-installer | bash

# 添加rbenv到环境变量
echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
source ~/.bashrc

# 安装Ruby 3.2.2
rbenv install 3.2.2
rbenv global 3.2.2

# 验证安装
ruby -v
```

#### 使用RVM安装Ruby
```bash
# 安装RVM
gpg --keyserver keyserver.ubuntu.com --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB
\curl -sSL https://get.rvm.io | bash -s stable
source ~/.rvm/scripts/rvm

# 安装Ruby
rvm install 3.2.2
rvm use 3.2.2 --default

# 验证安装
ruby -v
```

### 步骤3: 安装Node.js和Yarn

```bash
# 安装Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装Yarn
npm install -g yarn

# 验证安装
node -v
npm -v
yarn -v
```

### 步骤4: 获取项目代码

```bash
# 克隆项目（如果使用Git）
git clone <项目仓库地址>
cd yizhan

# 或者直接使用现有项目
cd /home/wjc/桌面/yizhan
```

### 步骤5: 安装项目依赖

```bash
# 安装Ruby依赖
bundle install

# 安装JavaScript依赖
yarn install

# 创建数据库
rails db:create
rails db:migrate

# 导入示例数据（可选）
rails db:seed
```

### 步骤6: 环境配置

#### 创建环境变量文件
```bash
# 创建.env文件
cat > .env << 'EOF'
# 应用配置
RAILS_ENV=development
RAILS_MAX_THREADS=5
DATABASE_URL=sqlite3:storage/development.sqlite3

# OCR配置
TESSERACT_LANG=chi_sim+eng
TESSERACT_PSM=6

# AI服务配置（可选）
DEEPSEEK_API_KEY=your_deepseek_api_key
ALIYUN_ACCESS_KEY_ID=your_aliyun_key
ALIYUN_ACCESS_KEY_SECRET=your_aliyun_secret

# Redis配置（如果使用）
REDIS_URL=redis://localhost:6379/0

# 文件上传配置
UPLOAD_MAX_SIZE=10485760
EOF
```

#### 配置Tesseract中文语言包
```bash
# 检查语言包安装
sudo apt install tesseract-ocr-chi-sim tesseract-ocr-chi-tra

# 验证语言包
tesseract --list-langs
# 应该显示: chi_sim, chi_tra, eng, osd
```

### 步骤7: 启动服务

#### 开发环境启动
```bash

rails server

# 启动Webpack开发服务器（端口3035）
./bin/dev

# 启动Sidekiq后台任务（如果使用Redis）

```

```bash
# 创建Tesseract配置文件
sudo nano /etc/tesseract/config

# 添加以下内容
tessedit_pageseg_mode 6
tessedit_ocr_engine_mode 3
load_system_dawg 0
load_freq_dawg 0
```

### ImageMagick优化
```bash
# 配置ImageMagick内存限制
sudo nano /etc/ImageMagick-6/policy.xml

# 修改内存限制
<policy domain="resource" name="memory" value="2GiB"/>
<policy domain="resource" name="map" value="4GiB"/>
<policy domain="resource" name="width" value="16KP"/>
<policy domain="resource" name="height" value="16KP"/>
```

## 🐛 故障排除

### 常见问题解决

#### 1. OCR识别失败
```bash
# 检查Tesseract安装
tesseract --version
tesseract --list-langs

# 重新安装语言包
sudo apt reinstall tesseract-ocr-chi-sim tesseract-ocr-chi-tra

# 测试OCR功能
echo "测试文字" | convert -size 300x100 xc:white -pointsize 24 -fill black -gravity center -annotate +0+0 @- /tmp/test.png
tesseract /tmp/test.png stdout -l chi_sim
```

#### 2. 图片上传失败
```bash
# 检查ImageMagick
convert --version

# 检查文件权限
ls -la public/uploads/
chmod 755 public/uploads

# 检查文件大小限制
cat config/application.rb | grep upload
```

#### 3. 依赖安装失败
```bash
# 清理并重新安装依赖
bundle clean --force
bundle install

# 清理JavaScript依赖
rm -rf node_modules
yarn install
```

#### 4. 端口冲突
```bash
# 检查端口占用
sudo netstat -tulpn | grep :3000

# 杀死占用进程
sudo kill -9 <PID>

# 或使用其他端口
rails server -p 3001
```
