## 🔧 集成架构设计

### 多服务降级机制
```ruby
class MultiOcrService
  def initialize(image_path)
    @image_path = image_path
    @services = [
      :paddle_ocr,      # 本地PaddleOCR (优先)
      :baidu_ocr,       # 百度OCR API
      :tencent_ocr,     # 腾讯OCR API
      :tesseract_ocr    # Tesseract (兜底)
    ]
  end
  
  def recognize
    @services.each do |service|
      begin
        result = send("recognize_with_#{service}")
        return result if result[:success]
      rescue => e
        Rails.logger.warn("#{service}识别失败: #{e.message}")
      end
    end
    
    { success: false, error: "所有OCR服务均失败" }
  end
end
```
